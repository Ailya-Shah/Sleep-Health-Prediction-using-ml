Ailya Shah

## Overview

This project analyzes sleep health data, builds machine learning models for prediction, and includes a Streamlit app for a quick binary sleep quality estimate.

## Files

- `sleep_health_analysis.ipynb`: notebook for analysis, modeling, and visualization
- `app.py`: Streamlit app for quick sleep quality prediction
- `sleep_health_dataset.csv`: dataset used in the project
- `requirements.txt`: Python dependencies for the app

## Run the App

```bash
pip install -r requirements.txt
streamlit run app.py
```
